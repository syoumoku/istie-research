# proxy_rules.md

## proxy 层级
1. 物业级直接证据
2. 同类物业 proxy
3. 城市级 proxy
4. 国家级 proxy

## 每条 proxy 必填字段
- proxy_type
- proxy_source
- proxy_scope
- proxy_distance
- proxy_penalty
- proxy_reason

## 证据约束层
以下字段不直接进入主评分：
- evidence_grade
- proxy_level
- hard_metric_completeness
- multi_source_check
- cap_reason
