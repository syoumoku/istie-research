# scenario_capacity_models.md

## 场景核心规模指标
- 机场：年旅客量
- 体育场馆：座位数
- 购物中心 / 大型零售综合体：ABL / GLA
- 会展中心 / 展览中心：展览面积 / 总面积
- 酒店：客房数
- 办公楼 / 总部园区：办公面积
- 医院：床位数
- mixed-use：主引擎面积 + 次引擎数量 + 园区复杂度

## Scale 分档
### 机场（年旅客量）
- 5：≥ 4000 万
- 4：2000–3999 万
- 3：1000–1999 万
- 2：300–999 万
- 1：< 300 万

### 体育场馆（座位数）
- 5：≥ 60000
- 4：40000–59999
- 3：20000–39999
- 2：10000–19999
- 1：< 10000

### 购物中心 / 大型零售综合体（ABL / GLA）
- 5：≥ 200000 ㎡
- 4：120000–199999 ㎡
- 3：70000–119999 ㎡
- 2：30000–69999 ㎡
- 1：< 30000 ㎡

### 会展中心（展览面积或总面积）
- 5：≥ 100000 ㎡
- 4：60000–99999 ㎡
- 3：30000–59999 ㎡
- 2：10000–29999 ㎡
- 1：< 10000 ㎡
