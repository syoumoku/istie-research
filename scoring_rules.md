# scoring_rules.md

## 五维主评分公式
Total Score = 20 × (0.24×PV + 0.28×CM + 0.18×PU + 0.18×BR + 0.12×CX)

## PV
目标：评估物业在城市和国家中的客观价值地位

## CM
目标：评估容量压力、峰值、驻留与热点复杂度

CM_raw = 0.40×Scale + 0.20×Peak + 0.15×Dwell + 0.15×Concentration + 0.10×Sensitivity

## PU
目标：评估高价值用户、商务用户、国际用户浓度

## BR
目标：评估品牌、口碑、门户、示范效应

## CX
目标：评估建筑和热点空间复杂度
